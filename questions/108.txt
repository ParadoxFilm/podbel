<code>
static public bool isEven(int ref a)
{
	if (a % 2 == 0)
		return true;
}
</code>
Какие ошибки были допущены программистом при объявлении метода <code>isEven</code>?
1) указан модификатор <code>static</code> перед модификатором <code>public</code>
2) тип параметра не соответствует типу возвращаемого значения
3) при некоторых значениях параметра оператор <code>return</code> не выполняется
4) имя метода начинается со строчной буквы
5) модификатор <code>ref</code> размещен после типа параметра
