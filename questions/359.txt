<code>void TakeMoney(decimal num, string password)</code>

1) <code>decimal TakeMoney(decimal num, string password)</code>
2) <code>public void TakeMoney(string password, decimal num)</code>
3) <code>static bool TakeMoney(decimal num, string password)</code>
4) <code>void GiveMoney(decimal num, string password)</code>
5) <code>void TakeMoney(decimal password, string num, string username)</code>
