Среди перечисленных утверждений выберите <b>верные</b>:
1) Структура не может служить возвращаемым значением
2) структура может применяться в качестве параметра, передаваемого по значению
3) структура не может служить параметром, снабженным модифиатором <code>ref</code>
4) структура не может служить параметром, снабженным модифиатором <code>out</code>
5) метод не может изменять значения полей структур, переданных в качестве параметра
