1;2;4;

Тип данных определяет:
1) объем памяти, выделяемой для размещения объекта этого типа
2) форму представления объекта этого типа в основной памяти
3) тип процессора, который должен быть использован для выполнения операций над объектом этого типа
4) набор операций, которые можно выполнять над объектом
5) форму представления объекта этого типа на экране
